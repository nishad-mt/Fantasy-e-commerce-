# Fantasy-e-commerce-
Fantasy is a full-stack food-based e-commerce web application built with Django.  
It allows users to browse products, add items to cart, place orders, and make payments, while providing an admin dashboard for managing products, categories, and orders.

This project was built as part of my learning and to demonstrate real-world backend and full-stack development using Django.

## Features

### User
- User registration & login (Email & Google)
- Product browsing & search
- Add to cart & wishlist
- Order placement
- Order history
- Profile management

### Admin
- Admin dashboard
- Product & category management
- Order management
- Offer & promotion management
- User management

---

## Tech Stack

- **Backend:** Django, Django Allauth  
- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Database:** SQLite (development)  
- **Authentication:** Email + Google OAuth  
- **Version Control:** Git & GitHub
